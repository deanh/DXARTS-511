Data Formats

Individual Download Record Format

Download data was pulled from the disparate sources and normalized in the following JSON format:

    {
      "referrer": {
        "url":"-",
        "host_ip":null,"host":"-"
      },
      "release":"Happy Birthday",
      "title":"Girls FM",
      "artist":"Happy Birthday",
      "format":"mp3",
      "source":"s3",
      "date":"2010-01-13",
      "request_location": {
        "city":null,
        "country_code":"US",
        "state":null,
        "lat":null,
        "lng":null
      },
      "time":"23:01:53"
    }

Summary Data Format

For the purposes of the specific visual approach used in this project, individual download records were reduced into summary JSON files which summarized per track downloads by date:
